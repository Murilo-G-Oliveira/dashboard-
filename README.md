# dashboard-

Este projeto consiste em um dashboard para análise de assinaturas do Xbox Game Pass, desenvolvido para monitorar métricas de vendas, planos contratados, add-ons adquiridos e o impacto de cupons de desconto nas assinaturas.

O dashboard permite visualizar de forma consolidada o desempenho financeiro e a distribuição dos clientes por tipo de plano.
